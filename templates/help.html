Accueil Historique Fleet Aide

Aide & Documentation

Navigation

  • Temps réel : Statut de la machine locale, actions rapides.
  • Historique : Courbes CPU/RAM/Disque, santé sur la durée.
  • Fleet : Vue centralisée de toutes les machines reportées.
  • Admin : Gestion des organisations, clés, tokens.

Actions rapides

  • Utilisez les boutons d’action pour nettoyer, redémarrer des services, collecter des logs.
  • Les tooltips (info-bulles) expliquent chaque bouton sensible.

Clés API & Sécurité

  • La clé admin est générée à l’installation : gardez-la secrète !
  • Les organisations et tokens se gèrent dans l’interface Admin.

Accessibilité

  • Navigation clavier : Tab/Entrée pour tous les boutons et champs.
  • Contraste renforcé, police lisible, responsive mobile.

Support

  • Consultez la documentation ou contactez l’administrateur pour toute question.
